# Yêu cầu hệ thống (System Requirements)
- PHP >= 7.4 (Khuyến nghị 8.0 trở lên)
- MySQL / MariaDB (Phiên bản mới nhất)
- Web Server: Apache hoặc Nginx (có thể sử dụng XAMPP, WAMP, MAMP, Laragon...)
- Composer (Trình quản lý thư viện của PHP)

# Yêu cầu Extension của PHP (PHP Extensions Required)
Các extension sau cần được bật trong file `php.ini` (thường đã được bật sẵn trên XAMPP):
- extension=pdo_mysql (để kết nối CSDL qua PDO)
- extension=mbstring  (để xử lý chuỗi Unicode)
- extension=fileinfo

# Cài đặt dự án (Setup Instructions)
Dự án PHP này quản lý các thư viện bên thứ 3 (như PHPMailer) thông qua Composer. Không giống Python sử dụng `pip install -r requirements.txt`, trong PHP chúng ta sử dụng file `composer.json`.

Để cài đặt phần phụ thuộc (dependencies) của project này, hãy mở Terminal/Command Prompt tại thư mục gốc của dự án và chạy lệnh:

    composer install

Sau khi chạy lệnh này, thư mục `vendor/` sẽ được tự động tạo và chứa tất cả mã nguồn bên thứ ba cần thiết để project hoạt động.
